Naive Bayes Classifer
=====================

## Introduction

This project implements Naive Bayes Classifer in C++.  It utilizes the
Thread Pool (http://github.com/weesan/thread/) for multi-processing
during training and testing.

## Installation

    $ configure
    $ make
