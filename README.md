k-means in C++
==============

## Introduction

This project implements k-means algorithm in C++.  It also comes with
a demo program using FLTK for animation.

## Dependencies

- [fltk-1.3.3](http://www.fltk.org/)

## Installation

    $ configure
    $ make

## Demo

    $ k-means 100000 5
